# Style Guide - Music Annotations App

## Typography

### Font Family
- **Primary Font**: Gaegu (all text)
- **Weight**: 700 (Bold) for titles and buttons
- **Transform**: UPPERCASE for titles and button text

### Usage
- Body text: Gaegu Regular
- Titles (h1-h6): Gaegu Bold + UPPERCASE
- Buttons: Gaegu Bold + UPPERCASE
- Form labels: Gaegu Bold

## Color Palette

### Primary Colors
```css
--dominant: #FFFBF0    /* Cream/Off-white - Main background */
--secondary: #DAF5D8   /* Light Green - Secondary backgrounds */
```

### Accent Colors
```css
--accent-dark: #525252  /* Dark Gray - Text, borders */
--accent-mint: #DDF3EB  /* Light Mint - Highlights, info */
--accent-pink: #FFDEDE  /* Light Pink - Alerts, emphasis */
```

## Component Styling

### Buttons
- Font: Gaegu Bold
- Text: UPPERCASE
- Border-radius: 8px
- Padding: 10px 20px
- Hover: opacity 0.85

### Forms
- Inputs/Selects: 2px solid border (--accent-dark)
- Border-radius: 5px
- Padding: 8px
- Background: white

### Headers
- Font: Gaegu Bold
- Text: UPPERCASE
- Color: --accent-dark

### Cards/Sections
- Background: white or --secondary
- Border-radius: 8px
- Padding: 20px
- Border: 2px solid when emphasized

## Color Usage Guide

- **Dominant (#FFFBF0)**: Page backgrounds, main canvas areas
- **Secondary (#DAF5D8)**: Section backgrounds, cards
- **Accent Dark (#525252)**: Text, borders, icons
- **Accent Mint (#DDF3EB)**: Success states, info boxes, neutral highlights
- **Accent Pink (#FFDEDE)**: Warnings, conflicts, important notices
